# Whooing Transformer

은행 및 카드사의 거래내역 데이터를 '후잉 가계부(Whooing)'의 외부입력 엑셀 형식으로 변환해 주는 웹 애플리케이션입니다.

## 주요 기능

- **우리은행**: 엑셀 파일 업로드 및 변환
- **하나은행**: 엑셀 파일 업로드 및 변환 (전처리 로직 포함)
- **쿠페이**: 앱 거래내역 텍스트 파싱 및 변환

## 기술 스택

- **Framework**: React (Vite)
- **Styling**: Tailwind CSS
- **Library**: `xlsx` (SheetJS)

## 설치 및 실행 방법

1. 저장소 클론
   ```bash
   git clone <REPOSITORY_URL>
   cd whooing-transformer
   ```

2. 의존성 설치
   ```bash
   npm install
   ```

3. 개발 서버 실행
   ```bash
   npm run dev
   ```

4. 브라우저에서 확인 (`http://localhost:5173`)

## 사용 방법

1. **금융사 선택**: 상단 탭에서 변환할 금융사(우리은행, 하나은행, 쿠페이)를 선택합니다.
2. **데이터 입력**:
   - **은행**: 엑셀 파일을 드래그 앤 드롭하거나 클릭하여 업로드합니다.
   - **쿠페이**: 거래내역 텍스트를 복사하여 입력창에 붙여넣습니다.
3. **변환 및 다운로드**: 'Convert & Download' 버튼을 클릭하여 변환된 엑셀 파일을 다운로드합니다.
4. **후잉 업로드**: 다운로드한 파일을 후잉 가계부 외부입력 기능으로 업로드합니다.
